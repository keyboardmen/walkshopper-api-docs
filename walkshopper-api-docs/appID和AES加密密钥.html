appID和AES加密密钥

获取移动设备在server端注册的唯一appID和AES加密密钥，用于用户后续注册和登录账号系统使用

URL（https格式）

https://123456.com

支持格式

JSON

请求方式

POST

请求参数

参数名 参数类型 是否必填 参数说明
appName string YES 标识app名称
uniqueID string YES 标识移动设备的唯一识别串号
appVersion string YES 用户当前使用的app版本
deviceType string YES 移动设备类型
systemName string YES 移动设备系统名称
systemVersion string YES 移动设备系统版本号

返回结果示例

{
	"retCode":200,
	"retDesc":"返回成功",
	"ret":{
		"appID":"2EE4FE1ED940B9FE1CB77425C651BD6AEDC2C3E0566780AE3795AEA8AE92958D56EF1BC4BCA66104804E0889C39A2A4B",
		"AESKey":"3aeef5f117788b4628cfb250c7f8a323"
	}
}

返回参数说明

参数名 参数类型 参数说明
appID string 标识当前移动设备的唯一ID
AESKey string AES加密所使用的密钥
