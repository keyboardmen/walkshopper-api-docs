#注册
用户注册接口。

##URL
http://115.28.228.41

##支持格式
JSON

##请求方式
POST

##请求参数
参数名  | 参数类型  | 是否必填  | 参数说明
:------------- | :------------- | :------------- | :-------------
username  | string  | Yes  | 用户注册的用户名，即手机号
passwd  | string  | Yes  | 用户的密码，需使用md5加密后传输

##返回结果示例
~~~
{
	"retCode":200,
	"retDesc":"返回成功"
}
{
	"retCode":450,
	"retDesc":"该手机号已经注册。"
}
~~~
