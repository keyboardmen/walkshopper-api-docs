自动登录接口

用于用户自动登录，验证token

URL

http://123456.com

支持格式

JSON

请求方式

POST

请求参数

参数名 参数类型 是否必填 参数说明
uniqueID string YES 标识当前移动设备的唯一ID
username string YES 用户注册的用户名，即手机号
loginToken string YES 用户的登录token

返回结果示例

{
	"retCode":200,
	"retDesc":"返回成功",
}

返回参数说明

参数名 参数类型 参数说明
loginToken string 表示用户已经登录的标识
