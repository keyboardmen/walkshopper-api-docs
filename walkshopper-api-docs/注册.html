注册

用户注册接口

URL（https格式）

https://123456.com

支持格式

JSON

请求方式

POST

请求参数

参数名 参数类型 是否必填 参数说明
appID string YES 标识当前移动设备的唯一ID
username string YES 用户注册的用户名，即手机号
passwd string YES 用户的密码，需使用md5加密后传输

返回结果示例

{
	"retCode":200,
	"retDesc":"返回成功"
}
{
	"retCode":450,
	"retDesc":"该手机号已经注册。"
}

返回参数说明

参数名 参数类型 参数说明
retCode int 200表示注册成功，从450开始表示注册失败
retDesc string 注册成功或注册失败的原因
